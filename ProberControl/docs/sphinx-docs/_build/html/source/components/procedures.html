procedures.raster.generate_map(xyz, p_meter, size, step)[source]¶

generates a map 3d map representing how much light is coupled over a specified region (size x step)

xyz (XYZ_Stage) p_meter (Power Meter) - instance size (2-tuple): the dimension of the region scanned step (int): the step size

returns a 2 dimensional list following the convention:
returned_map[3][10] = the “power”(actually voltage for now) at point (3,10)
